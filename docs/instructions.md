# Reviewer instructions

Thank you for reviewing SwiftMiner's detected Swift refactorings.

1. Open the detections catalogue and review any cases you choose.
2. Compare the before and after payloads and, when helpful, inspect the linked commit.
3. Submit whether the detection is correct, partially correct, incorrect, or uncertain.
4. Add a short note when the type or matched entities appear wrong.
5. You may return later or revise an earlier judgment.

Your reviewer code is pseudonymous. The same browser remembers it automatically.
To continue from another browser, bookmark or copy a review-page URL containing
your `reviewer_id`.
